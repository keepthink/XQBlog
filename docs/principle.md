> 本文档指明工作的原则.用来进行事务取舍.

####系统化思考
+ 用外在且俯视的眼光.将要做的行为拆建为优先的子系统.并逐渐优化. 

####提前计划
+ 每天晚上要计划第二天的事情.

####完全分类
+ 一个域问题.尽可能将所有的分类想全.方便机械化.

####N-1
+ 第一原理原则.思考问题的最本质

####20/80 原则
+ 只做关键的事情.拒绝打鼹鼠.从根源解决问题.
+ 精做关键事情的关键步骤

####取舍点
+ 出现不加思考的行为的时候.进行反省.

####输出大于输入
+ 输出大于输入.在实践中学习.在学习中实践.
